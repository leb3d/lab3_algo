## Лабораторна робота №3

**NumPy. Багатовимірні масиви**

Програма формує матрицю **3×4** випадкових чисел у діапазоні **(-50; 90)**.
Визначаються рядки з мінімальним та максимальним елементами, після чого виконується їх обмін.
Далі змінюється розмірність масиву на **6×2**, виводяться перші три рядки та вилучається останній рядок.

  Структура

* `generate_data()` — створення масиву
* `process_data()` — обробка
* `print_results()` — вивід
* `main()` — запуск програми

Використано базові можливості бібліотеки **NumPy**: генерація випадкових чисел, індексація, reshape та зрізи.
